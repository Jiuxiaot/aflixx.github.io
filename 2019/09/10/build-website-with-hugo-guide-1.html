Hugo建站指南1-使用Hugo构建个人博客

2019年9月10日
| 建站技术
| 阅读

本篇文章将指导你如何使用Hugo构建静态网站用于个人博客或者项目展示。

手把手教你如何从0开始构建一个静态网站，这不需要有太多的编程和开发经验和时间投入，也基本不需要多少成本（除了个性化域名），使用GitHub和Hugo模板即可快速构建和上线一个网站。

目标读者

本文档适用于以下用户：

对网站构建不太了解，没有web开发经验的用户 希望快速构建一个网站，个性化需求不多，不想花费太多精力打理，更新不多的用户 希望学习网站构建、网站模板、web开发的用户

Hugo简介

Hugo是由Steve Francis基于Go语言开发的静态网站构建工具，开源已4年，升级活跃，构建速度极快，开发者用户居多。

而使用静态网站构建工具不但可以让我们构建静态网站的工作大大简化，而且还可以迅速验证自己的想法，以最高效的方式开发网站。

步骤1：安装Hugo

Hugo的安装十分简单，安装时不需要安装任何依赖软件，其本身只是一个二进制文件，如果您是Mac用户，可以使用brew命令来安装，简单方便快捷。

brew install hugo

hugo version

以上命令能够显示hugo的版本，则说明hugo安装成功。

步骤2：创建站点

在你的mac选择你要存储的站点目录，执行如下命令：

hugo new site quickstart

则再当前目录生成quickstart的文件夹，也就是你的站点源文件目录。新创建的站点目录结构如下：

▸ archetypes/
▸ doc/content/
▸ data/
▸ layouts/
▸ static/
  config.toml

当前的站点没有任何内容，也没有进行任何配置。

步骤3：添加主题

创建完站点之后，由于没有添加主题，站点还不能显示出来。

Hugo 对皮肤支持的很好，有大量的现成皮肤可选，并且一直在增加中。 可以在themes.gohugo.io 中查看官方提供的主题列表。

要安装所有的 Hugo 皮肤，只需要 clone 完整的 hugoThemes 库到你的工作目录，这里演示一下安装Ananke theme主题。

cd quickstart

# Download the theme
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
# Note for non-git users:
#   - If you do not have git installed, you can download the archive of the latest
#     version of this theme from:
#       https://github.com/budparr/gohugo-theme-ananke/archive/master.zip
#   - Extract that .zip file to get a "gohugo-theme-ananke-master" directory.
#   - Rename that directory to "ananke", and move it into the "themes/" directory.
# End of note for non-git users.

# Edit your config.toml configuration file
# and add the Ananke theme.
echo 'theme = "ananke"' >> config.toml

在你的quickstart/themes目录下就可以查看到刚拉回来的主题ananke文件夹。

步骤4：添加文章

在你的quickstart目录下，执行创建文章命令hugo new posts/my-first-post.md。

这样新创建的文件my-first-post.md在目录 content/ 里，内容如下：

---
title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: true
---

步骤5：开启HugoServer

执行如下命令，运行Hugo Server。

hugo server -D

运行结果里提示，可以浏览器中打开http://localhost:1313/，这样就可以看到你的网站了。有一篇文章。

步骤6：配置你的网站

文本编辑器中打开 config.toml ，可以做如下配置，设置你站点的URL、标题、最终实用的主题等等。

baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

在你的第一篇文章my-first-post.md中，可以配置你的标题、文章时间、是否为草稿状态。然后添加你的内容吧。

---
title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: false
---

content
content
content

然后在运行hugo server，不加-D（是否显示草稿文章），http://localhost:1313/没有关掉的话会自动刷新页面。

再看一下，你的站点就有了标题，第一篇文章也有了内容了。
